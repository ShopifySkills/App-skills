# Complete the Look — One-Click Bundle Strategy (No Discount)

**Platform:** Shopify  
**Scope:** Foundation + Beauty Sponge + Setting Spray — convenience-only bundle.

---

## 1) Summary

- **Current gap:** No single “complete the look” offer; customers must add foundation, sponge, and setting spray one by one. Cross-sell is scattered or absent, so AOV and one-click convenience are left on the table.
- **Recommended bundle type(s):** One core bundle: **“Complete the Look — Base Set”** (foundation + beauty sponge + setting spray). No discount; value = one-click add and a clear “everything for the base” story.
- **Top 3 actions:**
  1. Define the bundle in Shopify (product bundle or app) with one hero (foundation) + sponge + setting spray; surface it on the foundation PDP above the fold.
  2. Add primary CTA “Add complete look to cart” and secondary “Add [item] only”; show bundle name, contents, and one-line value.
  3. Tag bundle orders and set up AOV, bundle attach rate, and PDP→cart conversion in analytics.
- **Metrics to watch:** AOV (all orders and bundle vs non-bundle), bundle attach rate (% of orders containing the bundle), PDP conversion (add-to-cart from pages where the bundle is shown).

---

## 2) Bundle Definition

- **Logic:** Group by *step* — base application (foundation + tool + finish). One hero product (foundation), two complements (sponge, setting spray).
- **Core bundle (name + SKUs + one-line value):**
  - **“Complete the Look — Base Set”** — [Your Foundation SKU] + [Your Beauty Sponge SKU] + [Your Setting Spray SKU] — *“Everything you need for a lasting base in one click.”*
- **Rules:** 3 items only; one hero (foundation), one tool (sponge), one finisher (setting spray). No discount; price = sum of components.

---

## 3) Naming and Copy

- **Bundle name:** **Complete the Look — Base Set** (or “Complete the Look — Base Kit”). Short, benefit-led; reuse everywhere (PDP, cart, analytics).
- **Primary CTA:** **“Add complete look to cart”** (or “Add set to cart” / “Get the set”).
- **Secondary CTA:** **“Add [Foundation / Sponge / Setting Spray] only”** so the single-item path is always visible.
- **Copy blocks:**
  - **Subhead:** “Foundation + sponge + setting spray. Everything for a flawless base in one click.”
  - **Bullet list:**  
    - “[Foundation] — [one benefit, e.g. buildable coverage]”  
    - “[Beauty Sponge] — [one benefit, e.g. seamless blend]”  
    - “[Setting Spray] — [one benefit, e.g. long-lasting finish]”  
    - “One click. One complete look.”
  - **Social proof (optional):** “Most customers add the complete look.”
- **Tone:** Match brand (clean/minimal or playful); outcome-focused (lasting base, easy application, one-click convenience). No medical/efficacy claims.

---

## 4) Placement and UX

- **PDP (above the fold):** Place the “Complete the Look — Base Set” block near the main product (foundation). Include:
  - Bundle name and one-line value.
  - List of 3 items with short benefit each.
  - **One-click:** Single “Add complete look to cart” (adds all 3).
  - **Single-item option:** “Add [Foundation] only” / “Add [Sponge] only” / “Add [Setting Spray] only” visible so the bundle is never forced.
  - Show bundle price = sum of components (no discount) and optional per-item breakdown.
- **Cart:** When the bundle is added, show a line such as “Part of Complete the Look — Base Set” with optional link to edit/remove from set.
- **Collection / landing (optional):** “Complete the Look” collection or section linking to this bundle (and any future bundles) for discovery.
- **One-click:** One action adds the whole bundle; separate controls for adding each item alone.

---

## 5) Pricing

- **No discount:** Bundle price = foundation + sponge + setting spray (sum of components). No percentage or dollar off.
- **Value message:** Emphasize convenience and clarity — “One click. One complete look.” — not savings.
- **Display:** Show total bundle price and, if helpful, per-item breakdown so the offer is transparent.

---

## 6) Metrics

- **Primary:**
  - **AOV:** All orders; segment AOV for orders that include the bundle vs orders that do not.
  - **Bundle attach rate:** % of orders (or checkouts) that include “Complete the Look — Base Set.”
  - **Conversion:** PDP → add-to-cart (or checkout) for pages where the bundle is shown; compare before/after or bundle vs non-bundle PDPs if possible.
- **Secondary:** Refund/return rate for bundle vs single-item orders (signal that the bundle isn’t driving misfits).
- **Signals:** If attach rate is low → test placement (higher on PDP) and copy (stronger “one-click” and benefit). If AOV rises but conversion drops → simplify messaging or reduce friction (e.g. one-click add).
